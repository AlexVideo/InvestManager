============================================
  Как собрать один самораспаковывающийся .exe
============================================

На ВАШЕМ компьютере (где есть Python и вы разрабатываете):

1. Установите зависимости (если ещё не стоят):
   pip install -r requirements.txt
   pip install pyinstaller

2. Установите 7-Zip (бесплатно): https://www.7-zip.org/
   Он нужен только для сборки — на целевом ПК не требуется.

3. Запустите скрипт сборки:
   build_sfx.bat

4. В папке dist появится файл:
   InvestManager_0.4_setup.exe

На целевом компьютере (где нет Python и архиваторов):

• Скопируйте туда только один файл InvestManager_0.4_setup.exe
• Запустите его — программа сама распакуется и предложит запустить Invest Manager
• Python, 7-Zip, WinRAR и т.п. ставить не нужно — после распаковки достаточно запустить InvestManager.exe из папки

Альтернатива без 7-Zip:
  Запустите: pyinstaller --noconfirm InvestManager.spec
  Скопируйте на другой ПК папку dist\InvestManager целиком и запускайте InvestManager.exe из неё (Python на том ПК не нужен).
